<h1 align="center">Contacts</h1>


## Описание
Консольное приложение «Контакты». Контакт представляет собой полное имя (fullName), номер телефона (phoneNumber) и адрес электронной почты (email).

## Команды для работы с контактами:

<b>close</b> либо <b>0</b> - Завершение работы приложения

<b>list</b> - Вывод всех имеющихся контактов в формате «Ф. И. О. | Номер телефона | Адрес электронной почты».

<b>Фамилия имя отчество; +79999999999; email@example.example</b>- Добавление новый контакт в список контактов. Формат ввода для обработки данных: Ф. И. О.; номер телефона; адрес электронной почты.

<b>delete</b> email@example.example - Удаление контакта по email.

<b>save</b> - Сохранение контактов в файл.


При установке в конфигурационном файле application.properties значения параметра spring.profiles.active=init, производится предварительная загрузка контактов из сайта, 
путь к которому определяется параметром app.default-contacts.

При значении параметра spring.profiles.active=default загружается пустой список контактов.
